# Lista de Tarefas (To-Do List)

Aplicação simples de tarefas com localStorage, validação de campo vazio e suporte à tecla Enter.

## Tecnologias usadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Adicionar tarefas (botão ou Enter)
- Marcar como concluída
- Excluir tarefa
- Salvar automaticamente no navegador (localStorage)

## Como executar
Basta baixar os arquivos e abrir o `index.html` no navegador.
